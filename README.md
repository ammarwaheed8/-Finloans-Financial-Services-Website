# 💰 Finloans — Financial Services Website

A responsive frontend website built for **Finloans**, a financial services company. Developed during my internship at **[Ezitech Institute](https://ezitech.org/), Pakistan**.

---

## 🌐 Live Preview

⚠️ This is a portfolio/demo version of the site. Some features are intentionally disabled — the fully functional version has been delivered to the client. Certain assets and functionality are restricted and are disabled for copyright reasons.

---

## 📌 About the Project

**Finloans** is a professional financial services website designed to present loan offerings, company information, and client contact options in a clean, modern interface. The site was built as **Project 3** during my frontend development internship at Ezitech Institute, Lahore, Pakistan.

The goal was to deliver a pixel-perfect, fully responsive website that reflects the brand identity of a real financial services company.

---

## 🛠️ Built With

| Technology | Purpose |
|------------|---------|
| **HTML5** | Page structure & semantic markup |
| **CSS3** | Styling, layouts, animations |
| **JavaScript (Vanilla)** | Interactivity & dynamic behavior |

> No frameworks or libraries were used — pure HTML, CSS, and JS.

## 🚀 How to Run Locally

Since this is a plain HTML/CSS/JS project, no build tools or installations are needed.

```bash
# Clone the repository
git clone https://github.com/ammarwaheed8/-Finloans-Financial-Services-Website.git

# Navigate into the folder
cd Finloans-Financial-Services-Website

# Open index.html in your browser
# (Double-click it, or use Live Server in VS Code)

```
💡 Recommended: Use the **Live Server** extension in VS Code for the best experience.
